# SEA-ECS-Cluster

Crée un cluster ECS pouvant accueillir des services ECS.

## Ressources créées

* aws_ecs_cluster

## Variables

| Nom | Description |
| --- | ----------- |
| identifier | Nom unique pour identifier les ressources AWS |


## Data sources 

| Nom | Description |
| --- | ----------- |
| cluster_arn | ARN du cluster ECS |
| cluster_id | Identifiant du cluster ECS |
